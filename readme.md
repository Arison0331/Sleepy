# **Sleepy**

一键释放浏览器页签占用内存

可设置有媒体播放时二次确认

可设置自动释放内存的时限，和自动关闭页签的时限

支持中英文切换

One-click release browser tab takes up memory

Can be set to confirm twice when there is media playback

Time limit for automatic memory release and time limit for automatic tab closure can be set

Support Chinese and English switching
